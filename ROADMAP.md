# PITS Development Roadmap

The project follows a 5-phase evolutionary path, increasing in complexity and sophistication.

## Phase 1: Institutional Foundation (Current)
- [ ] Core Orchestrator (Brain) implementation.
- [ ] Basic MT5 Data Ingestion.
- [ ] OFI (Order Flow Imbalance) calculation.
- [ ] Bayesian Inference model for direction.
- [ ] Fixed Kelly Criterion position sizing.

## Phase 2: Microstructure & Correlation
- [ ] OBI (Order Book Imbalance) 10-level depth.
- [ ] Trade Flow Toxicity (Aggressor side) detection.
- [ ] Multi-asset Neurons (Gold, DXY, Brent) integration.
- [ ] Cross-asset lead-lag features.

## Phase 3: Regime Detection & Memory
- [ ] Markov Switching Models for regime detection.
- [ ] Library of historical patterns (Pandemic, Geopolitics).
- [ ] Economic calendar integration (News API).
- [ ] Dynamic TP/SL based on market context.

## Phase 4: Machine Learning & Portfolio Optimization
- [ ] XGBoost & Random Forest implementation.
- [ ] Mean-Variance Optimization (Markowitz).
- [ ] Monte Carlo Risk Simulations (10,000 runs).
- [ ] Isolation Forest for anomaly/manipulation detection.

## Phase 5: GNN & Autonomous Evolution
- [ ] Graph Neural Networks (PyTorch Geometric) for asset relationships.
- [ ] Transformer models for temporal attention.
- [ ] Online Learning (self-calibrating models).
- [ ] Reinforcement Learning for strategy discovery.
